# Hi, I'm Jairo Torregrosa

**Senior AI Research Engineer at Mercado Libre** and **Claude Community Ambassador in Bogotá**, focused on Agentic Engineering.

I build the engineering layer around AI models: agents, tools, context, evals, traces, and the workflows that make AI systems useful in real software environments. My path into agentic systems came through evaluation. First model evals, then agent evals, now building systems that need to be measured before they can be trusted.

## Current focus

- Agentic engineering and long-running agent workflows
- Claude Code, skills, plugins, and developer-tool automation
- Evaluation harnesses for LLMs, agents, trajectories, and regressions
- Tool-use infrastructure, MCP-style integrations, and permission-aware interfaces
- AI for software engineering: review loops, CI feedback, repo automation, and coding agents
- Production-oriented AI systems: reliability, cost, latency, observability, and failure modes

## Selected work

- **[jaiskills](https://github.com/JairoTorregrosa/jaiskills)**: Claude Code skills and plugins, including Insistir, a multi-agent orchestration workflow where agents implement, review, revise, and judge work before it is accepted.
- **[SmolNews](https://github.com/JairoTorregrosa/SmolNews)**: A small agent running as a Hugging Face Space that fetches news and local time for a requested place.

Some applied research and product experiments are private while I clean up documentation, permissions, and public-safe examples.

## Background

Before GenAI, I worked on production machine learning, recommender systems, ranking, and computer vision. That background still shapes how I build AI systems: define the task, measure behavior, inspect failures, and keep the system grounded in real constraints.

I studied Electronic Engineering at Pontificia Universidad Javeriana and co-authored the IEEE STSIVA 2019 paper **"Comparing state-of-the-art methods of detection and tracking people on security cameras video."**

## Community

As a Claude Community Ambassador in Bogotá, I help bring together builders, founders, solopreneurs, and technical teams working with agentic engineering. The goal is practical: real products, real constraints, hands-on sessions, and shared learning around the Claude ecosystem.
